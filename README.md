# iFood Data Analytics Case

Autor: Andreza Castro

## Objetivo

Este projeto apresenta a solução de um case técnico de Data Analytics envolvendo:

- Análise de teste A/B de campanha de cupons
- Avaliação de impacto em métricas de negócio
- Análise de viabilidade financeira
- Segmentação de usuários
- Recomendações estratégicas

## Tecnologias utilizadas

- Python
- Pandas
- Google Colab
- Parquet

## Estrutura do projeto

notebooks/ → análise exploratória e respostas do case
src/ → scripts auxiliares de processamento
data_sample/ → amostra dos dados

## Como executar

1. Clone o repositório

git clone https://github.com/SEU_USUARIO/ifood-data-analytics-case.git

2. Instale as dependências

pip install -r requirements.txt

3. Execute o notebook

notebooks/ifood_case_analysis.ipynb

## Observação

O dataset completo não foi incluído no repositório devido ao tamanho.  
A análise foi realizada utilizando processamento em lotes para lidar com grandes volumes de dados (~3.6 milhões de registros).
